¿Cómo accede el recepcionista a su vista?

El recepcionista tiene su propio dashboard (pedidos activos, cambiar estado, alertas de cuenta). La pregunta es si vive dentro del admin o en una URL propia.

A — Sección dentro del sidebar del admin

El admin puede ver todo. En producción el recepcionista entraría con su propio usuario y vería solo su sección. Para el mock: el admin navega entre todas las secciones incluyendo la del recepcionista.

B — URL propia /recepcionista sin sidebar de admin

Vista completamente independiente, pensada para una tablet en la barra. Sin sidebar de admin — solo los pedidos. El admin llega ahí desde un link en su dashboard.
